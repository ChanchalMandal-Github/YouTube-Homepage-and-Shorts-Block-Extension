YouTube Homepage and Shorts Block Extension 

A Browser Extension designed to eliminate distractions on YouTube by enforcing a search-only homepage and preventing the "infinite scroll" trap of Homepage and Shorts.

Key Features:
1. Search-Only Home: Hides all video and recommendations on the homepage, leaving only the search bar.
2. Kill Shorts Scroll: Allows you to watch exactly one Short; if you attempt to scroll to a second one, you are automatically redirected to the Home page.
3. Hide YouTube Mixes: Removes "Mix" playlists.
4. Disable Infinite Scroll: Stops homepage results from loading infinitely .
5. Hide All Shorts: If you don't want any Short in your YouTube.

Privacy Policy

This extension does not collect, store, or transmit any personal user data. All data is stored locally on the user's device.
